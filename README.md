Computer Science student at the National and Kapodistrian University of Athens (NKUA). Currently completing my thesis on AR navigation and educational support systems.

### Thesis Tech Stack

| Category | Technologies |
|----------|--------------|
| **Mobile** | Kotlin, Android Studio, XML, ARCore/Google AR API, OpenMaps API, Firebase |
| **Backend** | Node.js, PHP, PM2 |
| **Database** | SQL, DBeaver, Firebase |

### Key Features
- 🗺️ **AR Navigation** - Cloud anchor-based guidance
- 📅 **Schedule Management** - Real-time schedule display
- 📍 **Location Services** - Real-time user presence recognition
- 🗄️ **Database Integration** - SQL backend
- 🔍 **Spatial Mapping** - Indoor/outdoor positioning
